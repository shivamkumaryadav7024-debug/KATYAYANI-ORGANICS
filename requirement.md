# Task Management Portal - MERN Stack

## Overview
Build a simple Task Management Portal using the MERN stack with AI assistance allowed.

**Requirements when using AI tools:**
- Share your AI prompts
- Explain your code clearly

## Core Features

1. **Add Task**
    - Title (required)
    - Description (optional)
    - Status: Pending / Completed
    - Created At (auto-generated)
    - Additional attributes as needed

2. **View Tasks**
    - Display all tasks in a list
    - Show status clearly
    - Show created date
    - Additional attributes as needed

3. **Mark Task as Completed**
    - Toggle status between Pending/Completed
    - Filter tasks: All / Pending / Completed

## Technical Stack

- **Frontend:** React (Vite)
- **Backend:** NestJS
- **Database:** MongoDB
- **Authentication:** Required
